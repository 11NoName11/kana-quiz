# Kana Quiz 🎌

Kana Quiz is a Japanese kana learning web app built with React.js.  
Practice reading and memorizing **Hiragana** and **Katakana** in a simple and interactive way.

🌐 Live Demo: https://quizjapan.vercel.app  
📦 Repository: https://github.com/11NoName11/kana-quiz

---

## ✨ Features

- Hiragana quiz
- Katakana quiz
- Fast and responsive UI
- Mobile-friendly design
- Built with React.js
- Progressive Web App (PWA) support

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/11NoName11/kana-quiz.git
```

### 2. Open Project

```bash
cd kana-quiz
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm start
```

App will run at:

```txt
http://localhost:8080
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Production files will be generated inside the `dist` folder.
